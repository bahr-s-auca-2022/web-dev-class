Welcome to My Project

We propose to create a dynamic and user-friendly website for the final project of the "Introduction to Web Development" course. The website's primary goal is to provide users with a comprehensive collection of movies that they can search through. When a user searches for a movie, the website will retrieve relevant movie information, including the movie's image, short description, IMDb score, and user ratings, from "The Movie Database" (TMDb) API.
